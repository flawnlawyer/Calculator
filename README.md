A modern, fully-functional calculator built with Python and Tkinter featuring a sleek UI, keyboard integration, memory functions, and dark mode.

## Features

✨ **Modern UI Design**
- Beautiful light and dark themes
- Smooth hover effects and animations
- Responsive layout that scales with window size
- Glow effects in dark mode

⌨️ **Full Keyboard Support**
- Number keys (0-9) and operators (+, -, *, /, .)
- Enter key for equals (=)
- Backspace to delete last character
- Escape key to clear (C)

🧮 **Calculator Functions**
- Multi-step arithmetic expressions
- Safe expression evaluation
- Error handling (division by zero, invalid syntax)
- Memory functions (M+, M-, MR, MC)

🎨 **Visual Effects**
- Button hover animations
- Dark mode toggle (🌙/☀️)
- Smooth color transitions
- Modern color palette

## Requirements

- Python 3.6 or higher
- Tkinter (usually included with Python)

## Installation

No additional packages required! Tkinter comes pre-installed with Python.

## Usage

Run the calculator:

```bash
python calculator.py
```

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| 0-9 | Number input |
| +, -, *, / | Operations |
| . | Decimal point |
| Enter | Calculate (=) |
| Backspace | Delete last character |
| Escape | Clear display |

### Memory Functions

- **MC** - Memory Clear: Clears stored memory
- **MR** - Memory Recall: Recalls stored value
- **M+** - Memory Add: Adds current value to memory
- **M-** - Memory Subtract: Subtracts current value from memory

### Dark Mode

Click the 🌙/☀️ button in the top-right corner to toggle between light and dark themes.
